======================
Multi images in events
======================

This module implements the possibility to have multiple images (A.K.A. image
gallery) for an event.

Installation
============

To install this module, you need to:

* Install `OCA/server-tools <https://github.com/OCA/server-tools>`_.

Usage
=====

To use this module, you need to:

* Go to *Marketing > Events > Create*.
* You can use the image you see in the title as the main image.
* You can go to the *Images* tab to add more.
